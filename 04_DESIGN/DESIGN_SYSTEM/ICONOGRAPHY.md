# ICONOGRAPHY

## Objective

Preparar la estructura profesional para definir el sistema iconografico de NeuroSports USA sin generar iconos todavia.

## Principles

- La iconografia debe apoyar funcion, claridad y coherencia visual.
- Debe integrarse con precision dentro del sistema general.
- Debe evitar decoracion innecesaria o ruido visual.

## Best Practices

- Definir nivel de detalle, geometria y consistencia.
- Separar uso funcional de uso explicativo.
- Mantener relacion con typography, inputs y buttons.

## Restrictions

- No generar iconos en esta fase.
- No definir estilos por moda visual.
- No mezclar lenguajes iconograficos incompatibles.
