# SPACING

## Objective

Preparar la estructura profesional para definir el sistema de espaciado del ecosistema visual de NeuroSports USA.

## Principles

- El espaciado debe reforzar jerarquia, orden y respiracion visual.
- Debe existir coherencia entre componentes, layouts y pantallas completas.
- Las decisiones de espaciado deben favorecer claridad y precision institucional.

## Best Practices

- Separar spacing interno, externo y estructural.
- Vincular espaciado a ritmo visual y legibilidad.
- Definir escalas que puedan reutilizarse en web y software.

## Restrictions

- No fijar valores definitivos en esta fase.
- No construir espaciados arbitrarios por componente aislado.
- No romper la relacion entre spacing y grid system.
