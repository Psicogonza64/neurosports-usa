# NeuroSports USA

## Descripcion breve

NeuroSports USA es un proyecto digital institucional en construccion para una experiencia web premium orientada a neuropsicologia, rehabilitacion cerebral y rendimiento cognitivo.

## Stack tecnologico

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ESLint

## Comandos principales

- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run build`
- `npm run start`

## Estado del proyecto

- Fase actual: Prototipo navegable del Home con sistema visual provisional.
- Estado tecnico: Compila correctamente y listo para preparacion de despliegue.
